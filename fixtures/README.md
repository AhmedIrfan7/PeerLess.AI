# Fixtures

Synthetic PDFs generated by `make_fixtures.py` (reportlab). No copyright — all content is fabricated.

| File | What it tests | Key issue |
|---|---|---|
| `clean_paper.pdf` | Baseline, should produce zero high/medium findings | None — well-written RCT with pre-reg |
| `grim_violation.pdf` | GRIM check | M=3.7, n=11 on integer 1–5 scale — mathematically impossible |
| `pvalue_inconsistency.pdf` | statcheck | t(18)=2.40 reported as p<.001 — recomputes to p≈.027 |
| `bad_citation.pdf` | Citation verifier | Reference 3 has DOI `10.9999/this-doi-does-not-exist` |

Each PDF has a matching `<name>.expected.json` describing minimum expected findings.

## Regenerate

```bash
python fixtures/make_fixtures.py
```

## Add new fixtures

1. Add a `make_<name>()` function in `make_fixtures.py`.
2. Add a matching `<name>.expected.json`.
3. Reference it in the relevant agent test.
