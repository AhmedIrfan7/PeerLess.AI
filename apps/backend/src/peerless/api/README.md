# FastAPI Routers — api/

This package contains all FastAPI route handlers for the PEERLESS.AI REST API.

All endpoints are versioned under `/api/v1`.

## Planned Routers

- `papers.py` — Upload and retrieve paper records
- `reports.py` — Fetch and update IntegrityReports
- `findings.py` — Approve / reject individual findings
- `health.py` — Health check endpoint
