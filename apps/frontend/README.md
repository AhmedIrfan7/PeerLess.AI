# Frontend — Next.js Application

This directory contains the Next.js 14 (App Router) + TypeScript + Tailwind CSS frontend for PEERLESS.AI.

## Structure

- `src/app/` — Next.js App Router pages and layouts
- `src/components/` — Reusable PascalCase React components
- `src/lib/` — API client, utilities, type definitions

## Running Locally

```bash
npm install
npm run dev   # http://localhost:3000
```

Or via Docker Compose:

```bash
docker compose up frontend
```
